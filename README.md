🍽️ Site Web Restaurant — Responsive Design
📌 Description
Ce projet est un site web de restaurant entièrement réalisé en HTML5 et CSS3.
L'objectif principal était de créer une interface moderne, élégante et surtout 100% responsive, c'est-à-dire adaptée à tous les types d'écrans : ordinateurs, tablettes et smartphones.

🛠️ Technologies utilisées
TechnologieRôleHTML5Structure et contenu des pagesCSS3Mise en forme, animations et responsive design

📱 Responsive Design
Le site s'adapte automatiquement à toutes les tailles d'écran grâce à :

Media queries CSS — pour ajuster la mise en page selon la largeur de l'écran
Flexbox / CSS Grid — pour des mises en page fluides et flexibles
Unités relatives (%, rem, vw, vh) — pour des tailles adaptatives
Images fluides — redimensionnement automatique selon le conteneur

📄 Sections du site

Header / Navigation — Menu de navigation fixe avec logo
Hero / Bannière — Section d'accueil avec image plein écran et slogan
À propos — Présentation du restaurant et de son histoire
Menu — Carte des plats organisée par catégories
Réservation — Formulaire de réservation en ligne
Contact — Adresse, téléphone, horaires et carte
Footer — Pied de page avec liens et réseaux sociaux

Projet réalisé dans le cadre d'un apprentissage du développement web front-end.
